ISTRUZIONI GIT (GitBash)
Trasforma la cartella corrente in una repository locale
git init  
Per fare un pull
git pull [link]

Per fare il push
Collega la repository locale a quella in remoto
git remote add origin "[link]"
Aggiunge un messaggio di commit
git commit -a
Scrivi il messaggio e poi ESC ed 
:wq
Come ultimo passaggio
git push origin

Per pushare in un altro ramo:
git push origin [ramo]

rendere il progetto compatibile con git flow tramite comando
-git flow init

-git 
ciao ciao
